Experiment:
        Visual Lexical Decision

Description:
        Purpose of this experiment is to measure how quickly participants
        can classify stimuli as being members or not. For each trial a sentence
        is presented on the screen. Participant's task is to hit a yes-button
        for a member-correct sentence and the no-button for an member correct sentence.
        Self-paced. Output: RT and correctness.

References:
        Rubenstein, H., Garfield, L., & Millikan, J.A. (1970). 
          Homographic entries in the internal lexicon. 
          Journal of Verbal Learning and Verbal Behavior, 9, 487­494.
